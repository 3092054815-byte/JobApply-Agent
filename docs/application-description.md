# 求职申请 Agent 项目说明

JobApply Agent 是一个面向求职者的智能求职申请助手。项目关注的核心问题是：求职者在申请不同岗位时，需要反复阅读 JD、提炼岗位要求、调整简历表达、撰写求职信并准备面试，但这些步骤往往耗时、重复，而且很难做到真正针对目标岗位。

用户输入一份简历和一个目标岗位 JD 后，系统会生成一份结构化申请包，包括岗位匹配报告、简历优化建议、定制求职信、面试准备问题和投递跟进清单。项目目前以 Web MVP 的形式实现，内置样例简历、样例岗位和 mock 模式，评审无需配置 API key 也可以完整体验核心流程。

系统采用多 Agent 协作流程。JD 解析 Agent 负责提取岗位职责、必需技能、加分技能和关键词；简历分析 Agent 负责整理候选人的技能、项目经历、教育背景和可证明证据；匹配推理 Agent 对比岗位要求和简历内容，生成可解释的匹配分数、已覆盖技能、能力缺口和风险提示；申请材料生成 Agent 基于匹配结果给出简历改写建议和求职信；面试准备 Agent 根据岗位要求与候选人短板生成面试题、回答提示和短期补强计划。

项目也保留了真实模型接入的工程边界：浏览器前端不直接保存或发送模型 API key，真实模型调用会通过本地 server 处理。这样既能保证 GitHub 展示时的易运行性，也为后续接入真实 LLM 留出了清晰接口。

我在设计中刻意保留了诚信边界：系统不会伪造项目、学历、证书或工作成果，而是帮助用户更好地组织真实经历、识别能力缺口，并有针对性地准备申请和面试。

## 项目亮点

- 解决求职申请中简历定制、JD 分析和面试准备重复耗时的问题。
- 将求职流程拆成多个职责清晰的 Agent，而不是只做一次普通文本生成。
- 输出结构化申请包，结果可以被页面展示、测试和后续扩展复用。
- GitHub 项目包含在线 Demo、README、流程图、样例输入输出、CI 和自动部署。
- 强调真实经历表达，不鼓励或生成虚假求职材料。
